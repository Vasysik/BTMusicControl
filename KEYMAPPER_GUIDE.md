# Настройка KeyMapper для BT Music Control

## Что нужно установить
- KeyMapper (бесплатно, Google Play)
- BT Music Control (наше приложение)
- Рут НЕ нужен

---

## Настройка шаг за шагом

### Шаг 1 — Открой KeyMapper
Нажми "+" снизу → "Новый маппинг"

### Шаг 2 — Выбери триггер
Нажми "Добавить триггер" → зажми нужную кнопку на телефоне.

**Варианты кнопок:**
- Кнопка громкости (Vol Up / Vol Down)
- Кнопка питания (двойное нажатие)
- Кнопка камеры (если есть)
- Кнопка поиска (если есть на японофоне)

### Шаг 3 — Настрой режим нажатия (опционально)
Можно выбрать:
- Одиночное нажатие
- Двойное нажатие
- Долгое нажатие

Это позволяет повесить несколько действий на одну кнопку.

### Шаг 4 — Добавь действие
"Добавить действие" → "Намерение (Intent)" → "Отправить broadcast"

Заполни поля:
```
Action (Действие): com.mygadget.action.NEXT_TRACK
Package (Пакет):   [оставь пустым]
Class (Класс):     [оставь пустым]
```

**Для каждой кнопки свой action:**
| Что делает       | Action                           |
|-----------------|----------------------------------|
| Следующий трек  | com.mygadget.action.NEXT_TRACK   |
| Предыдущий трек | com.mygadget.action.PREV_TRACK   |
| Play / Pause    | com.mygadget.action.PLAY_PAUSE   |

### Шаг 5 — Ограничения (рекомендую)
"Добавить ограничение" → "Bluetooth устройство подключено" → выбери свой сервер-телефон.

Так кнопки будут работать ТОЛЬКО когда BT соединение активно.

---

## Пример: Vol Down = следующий трек

1. Триггер: Vol Down (одиночное нажатие)
2. Действие: broadcast → com.mygadget.action.NEXT_TRACK
3. Ограничение: BT устройство подключено → [имя сервера]
4. Сохрани → включи маппинг (переключатель справа)

---

## Готовые настройки для японофона

Если у тебя японский телефон с дополнительными кнопками:

| Кнопка           | Действие BT Music             |
|-----------------|-------------------------------|
| Vol Up           | NEXT_TRACK                    |
| Vol Down         | PREV_TRACK                    |
| Vol Up (долго)   | PLAY_PAUSE                    |
| Кнопка камеры    | PLAY_PAUSE                    |

---

## Важно
- BT Music Control должен быть запущен (сервис работает в фоне)
- KeyMapper нужно разрешение "Специальные возможности" (Accessibility) — это не рут,
  просто стандартная системная настройка, без неё он не перехватит кнопки
- В KeyMapper есть встроенный гайд по включению Accessibility service
